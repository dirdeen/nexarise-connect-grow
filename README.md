# NexaRise

NexaRise is a Sierra Leone-focused employment, mentorship, career-development and verified workforce platform.

## Tech Stack

- React
- Vite
- TypeScript
- TanStack Router
- Tailwind CSS
- Supabase

## Local Development

Create `.env` with:

```bash
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-publishable-key
```

Run:

```bash
pnpm install
pnpm run dev
```

## Backend

Supabase powers authentication, database records, Row Level Security and CV storage.
